# Smart Attendance

A Flutter project for tracking attendance.

## Getting Started

To get started with this project, follow these steps:

### 1. Fork the Repository
Click the **Fork** button at the top right of this page to create a copy of this repository in your GitHub account.

### 2. Clone the Forked Repository
Clone your fork to your local machine:
```bash
git clone https://github.com/MbahLesky/Smart-Attendance.git
cd smart_attendance
```

### 3. Setup the Flutter Project
Ensure you have [Flutter](https://docs.flutter.dev/get-started/install) installed. Then run:
```bash
flutter pub get
```

### 4. Run the App (after selecting your device)
```bash
flutter run
```
